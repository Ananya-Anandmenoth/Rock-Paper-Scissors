# rock-paper-scissors
A simple Game Rock Paper Scissors made with JavaScript, HTML, CSS
Welcome to the Rock, Paper, Scissors game! This is a simple command-line game where you can play against the computer and test your luck and strategy. It's simple and very impressive version of the game Rock Paper & Scissors.

How to Play
Rules
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
Features
Simple and fun to play.
Play against the computer.
Randomized computer moves for unpredictability.
